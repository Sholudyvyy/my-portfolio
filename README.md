# My Personal Portfolio

A simple, minimalist personal portfolio website built with [Astro](https://astro.build).

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Integrations:** React

## Getting Started

1.  **Install dependencies:**

    ```bash
    npm install
    ```

2.  **Start the development server:**

    ```bash
    npm run dev
    ```

3.  **Build for production:**

    ```bash
    npm run build
    ```
